# 横断決定ログ（DECISIONS）

このファイルは、複数ワークストリームに影響する決定を時系列で一本化する唯一の場所（source of truth）です。

この public 正本世代は 2026-07-21 に履歴非継承で開始しました。旧世代から carry した公開可能な行は、同じ ID・同じ本文を verbatim で保持します。旧世代の全ログの部分ビューではなく、新世代で同じ判断をするために必要な決定を選んだ現行 SSOT です。

## 書き方のルール

- 追記只進。確定した決定を上書きで消さない。
- 各行に ID `YYYY-MM-DD-NN` と状態を持たせ、理由と却下案を残す。
- 決定IDを参照するときは日付部分を省略せず、各IDを`YYYY-MM-DD-NN`の完全形で書く。
- 改訂は、新しい行を追加し、旧行を `改訂→<新ID>` にし、新旧を双方向に接続する。
- `起案` / `保留` は未確定節に置く。`保留` は再開 trigger を明記する。
- 公開できない旧決定を同じ ID の redacted 版として作らない。必要な公開 interface と why は新しい ID で決める。
